## SWOT Analysis

  Generic methodological issue assessment and solving tool.

### Status

  * achieved 2018-12-17

### Milestone

  * beta

### File System

  * arc -> archives (zip)
  * bld -> build-in (svg)
  * doc -> documents (txt, jpg)
  * img -> images (png, svg)
  * lib -> libraries (js)
  * src -> sources (css, js)
